算术运算符对数值进行计算:

```java
System.out.println(10 / 4);
//此时的结果是两者都是整数，所以计算机也认为是整数，向下取整为2
System.out.println(10.0 / 4); 
//此时的结果为2.5
double d = (10 / 4);
System.out.println(d);
//此时的结果为2.0   2.0和2是不一样的，数据类型不一样
```

```Java
System.out.println(10 % -3);//此时的结果为1
System.out.println(-10 % -3);  //结果为-1
```

{: .box-note}
**Note:** 在Java中取模的本质是  a % b = a - a / b * b



```java
int i = 10;
i++;//自增相当于i = i + 1 
System.out.println("i=" + i);//结果为11
int j = 8;
int k = ++j;
//先自增，再赋值给k，此时j = 9,k = 9
int k = j++;
//先赋值，然后自增,此时k = 8,j = 9

```

!a 表示取反，若a为True，则结果为False

a^b 表示，a和b不同时，结果为True，否则结果为False

##### 短路（&&）与 和逻辑与（&）



```java
int a = 4;
int b = 9;
if (a < 1 && ++b <50) {
    System.out.println("ok");
    //如果第一个条件为False，后续的条件不在判断，此时b = 9
}
if (a < 1 & ++b < 50){
     System.out.println("ok");
    //如果第一个条件为False，那么后续的条件会执行并且判断
    // 此时b = 10
}
```

{: .box-note}
**Note:**短路的含义就是，后续的不在执行。一般使用短路与，效率高。



##### 短路或（||）和逻辑或（|）



```Java
int x = 5,y = 5;
if(x++ ==5 | ++y ==6){   
//x先比较在自增结果为true，y先自增在比较结果为true，都为true，结果为true，所以x = 11
   x = 11;
}
System.out.println("x="+x +"y="+y)
//输出的结果为x =11,y =6
```


##### 三元运算符

基本语法：条件表达式？ 表达式1：表达式2；

**一真大师**

规则：如果条件表达式为true，那么结果是表达式1；如果条件表达式为false，那么结果是表达式2.

```java
int a = 10;
int b = 99;
int result = a > b ?a++:b--;
//执行第二个，先返回在自减，返回结果为99
int result = a > b ?a++:--b;
//执行第二个，先自减在返回，返回结果为98
```

{: .box-note}
**Note:**可以转换为if -else 格式。

```java
//练习找到三个数中最大数
int max1 = n1 > n2 ? n1:n2;
int max2 = max1 > n3 ? max1:n3;
System.out.println("最大数为" + max2);
//改为一句话：
int max = (n1 > n2 ? n1:n2) >n3 ? (n1 > n2 ? n1:n2):n3;
```


##### 进制的转换：

二进制->十进制：    （十进制转二进制除以2，直到商为零）

**0b1011 = 1 * 2^0 + 1 * 2^1 + 1 * 2^3= 11**

八进制->十进制：     （十进制转二进制除以8，直到商为零）

**0234 = 4 * 8^0 + 3 * 8 ^1+ 2* 8^2 = 156**

十六进制->十进制：   （十进制转二进制除以16，直到商为零）

**0x23A = 10 * 16^0 + 3  * 16^1 + 2 * 16^2 = 570**

二进制->八进制：(低位开始每三位一组，转成八进制即可) 

<u>（八->二，每位转成三位二进制即可）</u>

**0b11010101 = 0325**

二进制->十六进制：(低位开始每四位一组，转成十六进制即可)

<u>（十六->二，每位转成四位二进制即可）</u>

**0b11010101 = 0xD5**

##### 位运算符（BItOperator）：

Java中一共有七个位运算（`&`、`|`、`^`、`~`、`>>`、`<<`、和`>>>`）

|  **按位与&**  |      **两位全为1，结果为1，否则为0**       |
| :-----------: | :----------------------------------------: |
| **按位或\|**  |    **两位有一个为1，结果为1，否则为0**     |
| **按位异或^** | **两位一个为0，一个为1，结果为1，否则为0** |
| **按位取反~** |             **0变成1，1变成0**             |

举例如下：

```java
public class BItOperator{
    public static void main(String[] args){
        System.out.println(2&3);  
//先得到2的补码，3的补码，按位与：
// 2的补码     00000000 00000000 00000000 00000010
// 3的补码     00000000 00000000 00000000 00000011
// 按位与      00000000 00000000 00000000 00000010   
//(此时是补码)原码也是如此，结果为2
        System.out.println(~-2);
//-2的原码     10000000 00000000 00000000 00000010   
//第一位为符号位，1表示负，0表示正
//-2的反码     11111111 11111111 11111111 11111101   
//负数的原码=除了符号位不变其他位取反
//-2的补码     11111111 11111111 11111111 11111110   负数的补码= 反码+1
//~-2         00000000 00000000 00000000 00000001   
//按位取反，0变1，1变0     为运算后的补码  结果为1
        System.out.println(~2);
// 2的补码     00000000 00000000 00000000 00000010
// ~2操作      11111111 11111111 11111111 11111101    运算后的补码 这是一个负数 
//运算后的反码  11111111 11111111 11111111 11111100     运算后的反码=补码-1
//运算后的原码  10000000 00000000 00000000 00000011      结果为-3
    }
}
```

| **>>**  | **算术右移：低位溢出，符号位不变，并用符号位补溢出的高位** |
| ------- | :--------------------------------------------------------: |
| **<<**  |             **算术左移：符号位不变，低位补0**              |
| **>>>** |             **无符号右移：低位溢出，高位补0**              |

举例如下：

```java
int a = 1 >> 2;//本质为1 / 2 / 2   结果为0
int c = 1 << 2;//本质为1 * 2 * 2   结果为4
int d = 4 << 3;// 4 * 2 * 2 * 2   结果为32
int e = 15 >> 2;//15 / 2 / 2      向下取整结果为3 
```



##### **※原码、反码和补码：**

**对于有符号的而言：**

1. **二进制的最高位是符号位：0表示正数，1表示负数。`（0->0，1->-）`**
2. **正数的原码。反码和补码都一样（三码合一）**
3. **负数的反码= 它的原码符号位不变，其它位取反（0->1,1->0）**
4. **负数的补码= 它的反码+1，负数的反码= 负数的补码 -1**
5. **0的反码和补码都是0**
6. **java中不存在没有符号的数。**
7. **计算机运行的时候都是以补码的方式来运算的。**
8. **当我们看运算结果时，要看他的原码。※**
